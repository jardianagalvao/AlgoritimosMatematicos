Menu de Algoritmos

Nesta página você poderá selecionar dentre os 6 algoritmos abaixo, basta acessar o menu de seleção:
  • Contagem: Dado um valor de N números de entrada em um conjunto de dados,
    é realizada a contagem de quantos valores inteiros existem nesse conjunto de dados
    entre o primeiro dado (inclusive) e N (inclusive).
  • Fibonacci: apresenta a sequência Fibonacci a partir de um número informado pelo usuário
  • MDC: Calcula o áximo Divisor Comum entre dois números quaisquer.
  • Número Primo: Verifica se o número informado é primo.
  • Ordenação com Quicksort: Ordena um array (listagem de números) usando o método Quicksort.
  • Somatório: Realiza a soma de um conjunto de números.
